⚙ Configuración — GitHub API rate limit ▾

Sin token, GitHub limita las consultas a 60/hora por IP a api.github.com (compartidas entre todos los que usen el portal desde la misma red/oficina). Con un token de acceso personal (no requiere ningún permiso especial, alcanza uno sin scopes) el límite sube a 5000/hora.

El token se guarda únicamente en el almacenamiento local de tu navegador — nunca se envía a ningún servidor que no sea api.github.com.
